Title: Data Cleaning with Copilot
Short Description: This project automates the cleaning of a messy sales dataset using Python and GitHub Copilot. The script loads the raw CSV file, standardizes column names, removes invalid values, handles missing data, and writes a clean, analysis-ready CSV file
How to run:
  _ Install dependencies: pip install pandas
  _ Run the cleaning script: python src/data_cleaning.py
After running, the cleaned dataset will be created automatically at: data/processed/sales_data_clean.csv
